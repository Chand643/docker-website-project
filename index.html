Docker Container Website Project

A simple website hosted inside a Docker container using Nginx

About This Project

This project demonstrates how a static website can be hosted inside a Docker container. The website is built using HTML and CSS, then served using an Nginx web server running inside Docker.

What is Docker?

Docker is a containerization platform. It allows us to package an application with all its required files, libraries, and dependencies into one lightweight container. This makes the application easy to run on any system.

What is a Container?

A container is an isolated environment where an application runs. In this project, the website runs inside an Nginx container. The container has its own web server and website files.

How This Website is Running

First, an HTML file was created. Then a Dockerfile was written using the Nginx image. Docker copied the website file into the Nginx web directory. After building the Docker image, a container was started and port 8080 of the VM was connected to port 80 of the container.

Project Workflow

1. Create index.html website file
2. Create Dockerfile
3. Use Nginx as base image
4. Build Docker image
5. Run Docker container
6. Access website using VM public IP and port 8080

Port Mapping

The container runs Nginx on port 80. The VM exposes the website on port 8080. So when we open http://VM_PUBLIC_IP:8080, the request goes to the Docker container.
